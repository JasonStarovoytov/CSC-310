This project is the implementation of a LLRB tree insert function

The starter code is based off of our original RB tree implementation
that we made in class; therefore, there are other functions in the 
program that are not used in this implementation (like all of the
deletion logic).

I commented out the old insert fix function and made a copy of it to
implement the LLRB fix. 

all the insertFix() function does is go over the 3 cases for LLRB trees
and works its way up the tree. In the end, it makes the root black, and
the tree is valid!!

The insertFix() function is the only function I modified, so the other methods
are unchanged.

I hope you enjoy this implementation!! :D
